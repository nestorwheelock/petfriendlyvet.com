================================================================
                    NOTAS TÉCNICAS DE IMPLEMENTACIÓN
================================================================

1. INTERACCIONES HTMX:
   - Tabla de niveles de stock: hx-get="/inventory/stock-table/"
     hx-trigger="load, every 60s" para actualizaciones en tiempo real
   - Búsqueda de productos: hx-get="/inventory/search/" hx-trigger="keyup
     changed delay:300ms" para búsqueda en vivo
   - Checkboxes de reorden: hx-post="/inventory/reorder/select/"
     para selección por lotes
   - Conteo de stock: hx-post="/inventory/count/save/"
     hx-trigger="change" para auto-guardado

2. ESTADO ALPINE.JS:
   - Selección tipo carrito para generación de OP múltiple
   - Entrada de conteo con navegación por teclado
   - Integración de escáner de códigos de barras
   - Validación de formularios para recepción

3. ESCANEO DE CÓDIGOS DE BARRAS:
   - Usar QuaggaJS o similar para escaneo basado en navegador
   - Soportar EAN-13, Code 128, códigos QR
   - Integración API de cámara móvil
   - Alternativa de entrada manual de SKU

4. FUNCIONES EN TIEMPO REAL:
   - WebSocket o SSE para actualizaciones de niveles de stock
   - Notificaciones de alerta por stock bajo
   - Progreso de conteo en vivo para conteos en equipo

5. REPORTES:
   - Generación PDF para documentos de OP
   - Exportación Excel para reportes de stock
   - Reportes de auditoría de sustancias controladas

6. OPTIMIZACIÓN MÓVIL:
   - Botones grandes amigables al tacto
   - Navegación por deslizamiento para entrada de conteo
   - Capacidad offline para conteos de stock
   - Interfaz con cámara primero para escaneo

7. PERMISOS:
   - Ver inventario: todo el personal
   - Ajustar stock: personal + gerente
   - Aprobar conteos: solo gerente
   - Sustancias controladas: solo personal licenciado
   - Crear OP: gerente + propietario

8. PISTA DE AUDITORÍA:
   - Todos los movimientos registrados con usuario, marca de tiempo
   - Sustancias controladas con doble registro
   - Discrepancias de conteo requieren explicación
   - Flujo de aprobación de OP

ESTRUCTURA DE DATOS:
- Mascota pertenece a Propietario (Usuario)
- Propietario puede tener múltiples mascotas
- Mascota tiene:
  - Info básica (nombre, especie, raza, sexo, fecha nacimiento, color)
  - Info médica (historial peso, alergias, condiciones, microchip)
  - Registros médicos (visitas, diagnósticos, tratamientos)
  - Vacunaciones (con seguimiento de vencimiento)
  - Documentos (certificados, resultados de lab, imágenes)
  - Notas internas (solo personal, no visible para propietario)

PERMISOS:
- Propietario: Ver propias mascotas, reservar citas, subir documentos
- Personal: Todos los permisos del propietario + agregar registros médicos, notas internas
- Veterinario: Todos los permisos del personal + diagnosticar, recetar, aprobar registros

ACTUALIZACIONES EN TIEMPO REAL:
- Usar HTMX para cambio de pestañas (sin recarga completa de página)
- Alertas de vacunación se actualizan automáticamente
- Gráfica de peso usa librería de gráficos ligera (Chart.js o similar)

MANEJO DE DOCUMENTOS:
- Procesamiento OCR/Visión para documentos subidos (extraer texto de imágenes)
- Generación de PDF para certificados
- Almacenamiento seguro de archivos con URLs firmadas

ACCESIBILIDAD:
- Navegación por teclado para todos los elementos interactivos
- Etiquetas ARIA para indicadores de estado
- Colores amigables para daltonismo (usar iconos también)

OPTIMIZACIONES MÓVILES:
- Desplazamiento horizontal para pestañas en pantallas pequeñas
- Secciones colapsables para contenido largo
- Objetivos táctiles amigables (mínimo 44px)
- Gestos de deslizamiento para navegación de pestañas

INTEGRACIONES:
- Exportación de calendario (iCal/Google Calendar para citas)
- Exportación PDF del historial médico
- Compartir con veterinario externo (generación de enlace seguro)

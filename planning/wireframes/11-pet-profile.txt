DATA STRUCTURE:
- Pet belongs to Owner (User)
- Owner can have multiple pets
- Pet has:
  - Basic info (name, species, breed, sex, DOB, color)
  - Medical info (weight history, allergies, conditions, microchip)
  - Medical records (visits, diagnoses, treatments)
  - Vaccinations (with expiry tracking)
  - Documents (certificates, lab results, images)
  - Internal notes (staff only, not visible to owner)

PERMISSIONS:
- Owner: View own pets, book appointments, upload documents
- Staff: All owner permissions + add medical records, internal notes
- Vet: All staff permissions + diagnose, prescribe, approve records

REAL-TIME UPDATES:
- Use HTMX for tab switching (no full page reload)
- Vaccination alerts update automatically
- Weight chart uses lightweight charting library (Chart.js or similar)

DOCUMENT HANDLING:
- OCR/Vision processing for uploaded documents (extract text from images)
- PDF generation for certificates
- Secure file storage with signed URLs

ACCESSIBILITY:
- Tab navigation for all interactive elements
- ARIA labels for status indicators
- Color-blind friendly status colors (use icons too)

MOBILE OPTIMIZATIONS:
- Horizontal scroll for tabs on small screens
- Collapsible sections for long content
- Touch-friendly tap targets (44px minimum)
- Swipe gestures for tab navigation

INTEGRATIONS:
- Calendar export (iCal/Google Calendar for appointments)
- PDF export for medical history
- Share with external vet (secure link generation)

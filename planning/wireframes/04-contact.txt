1. CONTACT FORM HANDLING:
   ```python
   class ContactMessage(models.Model):
       name = models.CharField(max_length=200)
       email = models.EmailField()
       phone = models.CharField(max_length=20, blank=True)
       subject = models.CharField(max_length=50, choices=SUBJECT_CHOICES)
       message = models.TextField()

       # Tracking
       created_at = models.DateTimeField(auto_now_add=True)
       is_read = models.BooleanField(default=False)
       replied_at = models.DateTimeField(null=True)
       replied_by = models.ForeignKey(User, null=True)

       # Source tracking
       source = models.CharField(max_length=20)  # 'website', 'ai_chat', 'whatsapp'
       ip_address = models.GenericIPAddressField(null=True)
       user_agent = models.TextField(blank=True)
   ```

2. HTMX FORM SUBMISSION:
   ```html
   <form hx-post="/contact/submit/"
         hx-target="#contact-form-container"
         hx-swap="innerHTML"
         hx-indicator="#form-spinner">
   ```
   - Form submits via HTMX without page reload
   - Success state replaces form
   - Error messages appear inline
   - Honeypot field for spam prevention

3. GOOGLE MAPS INTEGRATION:
   - Use Google Maps Embed API (no API key required for basic embed)
   - Or use iframe with place search
   - Include "Get Directions" links that open in Google Maps app
   - Lazy load map on scroll into view

4. CLICK-TO-CALL/WHATSAPP:
   ```html
   <a href="tel:+529981234567">Call</a>
   <a href="https://wa.me/529981234567?text=Hola...">WhatsApp</a>
   ```

5. FAQ ACCORDION:
   - Alpine.js for expand/collapse
   - Schema.org FAQPage markup for SEO
   - Track which FAQs are clicked (analytics)

6. MOBILE OPTIMIZATIONS:
   - Sticky call/WhatsApp buttons at top
   - Form fields use proper input types (tel, email)
   - Large tap targets (min 44px)
   - Map opens in native app option

7. SPAM PREVENTION:
   - Honeypot hidden field
   - Rate limiting (max 3 submissions per IP per hour)
   - reCAPTCHA v3 invisible (optional)
   - Server-side validation

8. ACCESSIBILITY:
   - Form labels properly associated
   - Error messages linked to fields with aria-describedby
   - Focus management after form submission
   - Skip link to main content

9. SEO:
   - Schema.org LocalBusiness markup
   - NAP (Name, Address, Phone) consistency
   - Geo coordinates in meta
   - hreflang for ES/EN versions

10. ANALYTICS TRACKING:
    - Track form submissions
    - Track click-to-call events
    - Track WhatsApp clicks
    - Track FAQ expansions
    - Track map interactions
